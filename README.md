## 🚀 How to Use

### Basic Usage
1. Open `index.html` in your web browser
2. The website will load with all features enabled
3. Click navigation links to scroll to different sections
4. Use the contact form to set up messaging

### Customization

#### Modify Personal Information
Edit the following in `index.html`:
- Name and title in hero section
- About section content
- Contact information (phone, email)
- Social media links (update href attributes)

#### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2d7d9f;      /* Main blue color */
    --secondary-color: #0a5d7a;    /* Dark blue */
    --accent-color: #ff6b6b;       /* Red accent */
    --light-bg: #f8fafb;           /* Light background */
}
```

#### Update Content
All text content can be easily updated:
- Experience descriptions in the timeline
- Education information
- Skills and achievements
- Contact details

## 🔧 Customization Guide

### Adding New Sections
1. Add a new `<section>` in HTML with appropriate ID
2. Add navigation link in the navbar
3. Create CSS styling in styles.css
4. Add any necessary JavaScript in script.js

### Changing Animations
- Modify animation durations in `styles.css`
- Look for `--transition` variable and `@keyframes` animations
- Adjust timing and effects as desired

### Contact Form Integration
Currently, the form shows a success message. To actually send emails:
1. Use a service like Formspree, Netlify Forms, or similar
2. Add form action attribute: `<form action="YOUR_SERVICE_URL" method="POST">`
3. Update form submission handling in `script.js`
